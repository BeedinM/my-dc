Projeto de um jogo [DungeonCrawler] criado utilizando Next.js

## Iniciando o projeto

Primeiro, baixe todas as dependências com npm. Após isso, inicie o projeto:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
```
